# Time Slicer

<img width="1067" height="941" alt="image" src="https://github.com/user-attachments/assets/b9c7d7ae-0357-4a38-b27e-35bd9ed3c0e0" />


GUI tool for making image collages. Great for timelapses or comparisons

## Features
- add files or folders
- drag and drop to add files
- reorder files to change positions
- Customize the look
   - blend 
   - angle
   - drop shadow (forward/backward + alpha)
- .png export
